====== PDF ======

''sudo aptitude install evince''

В KDE изначально основной просмотрщик — //Okular,// в Gnome — //Evince//. Мне нравится второй по нескольким причинам:
* проще выглядит, почти всё спрятано за отдельными кнопками
* занимает меньше памяти, нежели Okular, который поначалу прикидывается нормальным, а после скролла в больших документах начинает нажирать мегабайты памяти. 435 345 

			Okular caches rendered pages in memory. Large documents can use a significant amount of RAM (but will never saturate available memory). Basically, Okular is using a lot of RAM because you have a lot of (spare) RAM. This behavior can be tuned in the settings: go to Configure > Configure Okular > Performance and check the options under "Memory usage" (Быстродействие > Использование памяти = Экономное). You can set it to "Low" to minimize memory usage at the cost of slower scrolling.
