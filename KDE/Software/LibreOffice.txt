====== LibreOffice ======

В системе может быть одновременно установлено несколько версий LibreOffice.

* скачать стабильную версию для своего ноутбука. 
	1. Открыть страницу загрузки https://www.libreoffice.org/download/download-libreoffice/
		a. или сразу пройти по https://www.libreoffice.org/download/download-libreoffice/?type=deb-x86_64&lang=ru
	2. справа в ссылках тыкнуть последовательно по «Choose operating system» и «Choose language». 
	3. выбрать версию подревнее. Не надо увлекаться «самой свежей версией», стабильность важнее.
	4. дважды убедиться в том, что к загрузке предлагается именно deb.
	5. тыцнуть по кнопке Donwload. Загрузится страница загрузки, скачивание начнётся автоматически через несколько секунд.
	6. на странице загрузки также появится кнопка для скачивания ’langpack_ru’ — его называют «//Translated User Interface//»

Аккуратно: если скачал deb, то и лангпак должен быть для deb.

Получить файлы вроде 
* LibreOffice_7.3.7_Linux_x86-64_deb.tar.gz
* LibreOffice_7.3.7_Linux_x86-64_deb_langpack_ru.tar.gz

* удалить нынешнюю версию LibreOffice, иначе в системе появятся две версии LibreOffice
	sudo > зайти в каталог /opt/ и удалить там каталог "LibreOffice_СТАРАЯ_ВЕРСИЯ"
	
* переместить оба файла в отдельный каталог. Распаковать их по-отдельности.
1. сперва основной
2. затем лангпак

Можно так:
	''tar -xvzf *.tar.gz''
	
В обоих случаях появятся новые каталоги с распакованными файлами.

переместить все файлы лангпака (из папки DEBS) в папку DEBS основного набора.
	
* перейти в распакованную "DEBS" directory
''sudo dpkg -i *.deb''

В каталоге /opt/ появится подкаталог с названием распакованной версии LibreOffice ("/opt/libreoffice7.3", например)

* настроить открытие файлов через: Открыть с помощью > Другое > Офис > «LibreOffice 7.3»
	
См. [[+Настройки LibreOffice]]
