Меня зовут Алексей Лупан, я головний спiвпрацювальник по підготувальні тестувальників în Moldova. Неспешный волговод, на моих ноутах стоит Debian, на полке книги и виниловые пластинки. Играю на баритон-укулеле лёгкий блюз, похожий на тяжелый.

	{{../images/alexei_lupan.jpg}} 

Здесь собраны мои заметки и инструкции по настройке рабочего стола KDE а также программного обеспечения в операционной системе [[Debian]] (иногда в [[Manjaro]]), которыми я пользуюсь ежедневно. Если кому-то ещё это всё будет полезно, то велком.

Исходные файлы проекта находятся в https://github.com/testitquickly/bystro.linux и постоянно используются в [[https://github.com/zim-desktop-wiki/zim-desktop-wiki|Zim]] — по-моему, это лучший десктоп-редактор для wiki-pages, который основан на безопасных plain-text файлах с markdown разметкой. Здесь находится автоматический экспорт в html из общего блокнота.

Ещё есть https://github.com/testitquickly/bystro.linux.nas — аналогичный сборник инструкций для управления NAS на основе OMV, его я не экспортировал.

	* NAS — Network Attached Storage — любое устройство, которое хранит файлы с доступом к ним по сети. 
	* OMV — OpenMediaVault, свободное ПО для NAS на основе Debian.

===== Метки =====

Метки (они же тэги) в начале каждого документа в версии html смотрятся как излишнее декорирование, но внутри Zim они автоматически собираются в облако тэгов и здорово помогают в упорядочивании содержимого.

===== Как открыть «bystro.linux» в Zim =====

==== Установить zim ====

* [[Debian:Установка ПО:zim]]
* [[Manjaro:Установка ПО:zim]]

==== Снять файлы из  git ====

https://github.com/testitquickly/bystro.linux/

==== Открыть в Zim файл „notebook.zim” ====

Также можно добавить новый блокнот во внутренний список блокнотов Zim:

''Открыть Zim''
''> File''
''> Open Another Notebook (Ctrl+O)''
''> Add''
''> [Browse]''
''> Указать каталог с этими файлами.''

Подхватится файл notebook.zim, блокнот откроется.
