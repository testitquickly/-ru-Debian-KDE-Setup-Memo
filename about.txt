===== Что такое «bystro.linux» =====

< @main @about >

Меня зовут Алексей Лупàн, я «головний спiвпрацювальник по підготувальні тестувальників în Moldova» — https://testitquickly.com/about/ 
Интересуюсь сложными задачами по тест-дизайну, баритон-гитарой и книгами Роберта Хайнлайна.

Этот проект — сборник инструкций для настройки KDE и сопутствующего ПО в Debian или Manjaro. Он собран в [[https://github.com/zim-desktop-wiki/zim-desktop-wiki|Zim]] — фантастише мультиплатформед-десктоп-текст-редактор wiki-pages на основе plain-text файлов с markdown разметкой. 

Проект находится в https://github.com/testitquickly/bystro.linux и экспортируется в html на https://testitquickly.com/bystro.linux/index.html

Ещё есть https://github.com/testitquickly/bystro.linux.nas — аналогичный сборник инструкций для управления NAS на основе OMV.

	* NAS — Network Attached Storage — любое устройство, которое хранит файлы с доступом к ним по сети. 
	* OMV — OpenMediaVault, свободное ПО для NAS на основе Debian.

===== Условности bystro.linux =====

Все инструкции расположены по разделам:

* [[Common]]
* [[Debian]]
* [[Manjaro]]

Навигация по документу выстроена через индекс страниц — https://testitquickly.com/bystro.linux/index.html (она же полная иерархя инструкций в проекте). В каждый головной файл каждого раздела должны быть добавлены ссылки на все страницы/подстраницы.

Для отображения в браузере все файлы экспортируются из Zim в html нативным способом. Дефолтный шаблон более удобен для просмотра с большого экрана.

Тэги в начале каждого документа в версии html смотрятся как излишнее декорирование, но внутри Zim они автоматически интегрируются в облако тэгов, к которых линкуются соответствующие файлы.

===== Открыть bystro.linux в Zim =====

==== Установка zim ====

* [[Debian:Установка ПО:zim]]
* [[Manjaro:Установка ПО:zim]]

==== Снять файлы ====

1. Выкачать все файлы из https://github.com/testitquickly/bystro.linux/
2. Открыть в Zim файл „notebook.zim”.

Также можно добавить новый блокнот во внутренний список блокнотов Zim:

''Открыть Zim''
''> File''
''> Open Another Notebook (Ctrl+O)''
''> Add''
''> [Browse]''
''> Указать каталог с этими файлами.''

Подхватится файл notebook.zim, блокнот откроется.
