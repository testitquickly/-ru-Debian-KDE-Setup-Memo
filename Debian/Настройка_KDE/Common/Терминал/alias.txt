====== alias ======

Любые длинные команды в консоли можно заменить алиасом (alias) — короткое слово, которое является переменной.

===== Объявить алиас =====

''alias nas='ssh'' [[astenix@192.168.80.100|''astenix@192.168.80.100'']] ''-i /home/ssh_keys/nas/nas_astenix'''

Теперь вместо длинной команды для подключения к удалённой машине по ssh достаточно набрать команду “''nas''”.

Алиас будет действовать до тех пор, пока открыта консольная сессия.

===== Сохранить алиас =====

Чтобы сохранить алиас «навсегда», его надо записать в последние строки файла ''~/.bashrc''

Алиас будет действовать только в профиле, в котором находится данный файл.

На самом деле происходит вот что: при загрузке профиля система «с нуля» читает содержимое ''~/.bashrc'' и выполняет прописанные там инструкции. Если файл удалить или изменить, то при следующей загрузке профиля инструкции снова будут выполняться «с нуля».

Чтобы изменения применились, надо «перечитать» файл ''~/.bashrc'' — или полностью перезагрузить профиль, или выполнить в консоли принудительное «прочитывание» файла:

''source ~/.bashrc''
