====== Удаление файлов и каталогов ======

Для удаления каталогов используется команда “rmdir”. Удалять можно только пустые каталоги, т. е. не содержащие файлов и поддиректорий (вложенных папок).

Для удаления файлов используется команда “rm”:

''rm test.txt''

Чтобы удалить не пустой каталог можно использовать команду rm с ключом “-r” (recursive). 

	При этом при удалении каждого вложенного объекта будет требоваться подтверждение.
	А если добавить ключ ”-f”, то вопросов на подтвержение не будет.

''rm -rf /mnt/save/alldocuments/''
