====== Поменять часы в трее на виджет Event Calendar ======

По тулбару правой кнопкой, Edit Panel или сразу Add widgets > появится выдвижная панель с виджетами. 

Кликнуть по кнопку "Get new widgets".

А затем на строке Получить новые виджеты Plasma...

Когда откроется Программа установки дополнений для Plasma, введите в Поиск 'event calendar' (календарь событий) и нажмите Установить.

	Вероятно, его там не будет.
	Тогда его можно установить из https://github.com/Zren/plasma-applet-eventcalendar

По окончании установки виджета кликните правой кнопкой мышки на часах в трее, а затем на строке 'Взаимозаменяемые виджеты'.

И далее из открывшегося перечня установленных виджетов выберите Event Calendar и кнопка Заменить.
Теперь вновь правый клик на часах и в открывшемся перечне Цифровых часов выберите Настроить виджет «Event Calendar».

**Шрифт часов**
* Chivo Black (толсто, с засечками, лучше всех)
* Open Sans Extra Bold (очень толсто, без засечек, красиво)
* Dante (толсто, вроде терминала, красиво)
* Helvetica (тонко)
* Clear Sans (тонко)

Как убрать информацию о погоде: Настроить виджет «Календарь событий» > Agenda > 
* Day spacing = 0px
* Event spacing = 0px
