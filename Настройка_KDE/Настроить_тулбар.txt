====== Настроить тулбар ======
А заодно и всякое другое хором.

===== Заменить виджет с иконкам на виджет с окнами =====
'''
Кликнуть правой кнопкой мыши по свободному месту на тулбаре 
> Взаимозаменяемые виджеты
> Выбрать виджет с названием «Панель задач»
> [Заменить]
'''

===== Настройка виджета «Панель задач» =====
'''
Правой кнопкой мыши по иконке любого открытого приложения (тот же Dolphin) 
> Дополнительно 
> Настроить виджет «Панель задач»
> Внешний вид
> Основное
'''
* Убрать галочку «Показывать всплывающие подсказки»
* Убрать галочку «Подсветка окна при наведении на соответствующий элемент в панели задач
* Поставить галочку «Помечать приложения, которые воспроизводят звук»
* Поставить галочку «Не начинать новый столбец, пока он не заполнится максимально»
* [Применить]

''> Поведение''
* Группировка: Не группировать
* Сортировка: Вручную
* Поставить галочку «Запретить перемешивать с кнопками запуска»
* Убрать галочку «Колесо мыши: Переключение между приложениями»
* [Применить]

===== Настроить отображение окон =====
'''
Параметры системы 
> Рабочая среда 
> Поведение рабочей среды 
> Эффекты 
> справа от панели «Поиск» кликнуть по кнопке «Настроить фильтр»
'''

* убрать галочку "Скрыть внутренние эффекты" 
* в разделе «Внешний вид» убрать галочку "Подсвечивать окно"
* [Применить]

===== Подавить превью окон в тулбаре =====
'''
Параметры системы 
> Рабочая среда 
> Поведение рабочей среды 
> Основные параметры
> Визуальное поведение
'''
* Отключить галку «Всплывающие подсказки при наведении курсора мыши»

===== Настроить виртуальные столы =====
'''
Параметры системы 
> Рабочая среда
> Поведение рабочей среды 
> Рабочие столы
'''

* [Добавить] новый рабочий стол 
* Настроить параметры перехода и анимации
* [Применить]

На тулбаре появится иконка двух рабочих столов.

Пр''авой кнопкой по этой иконке ''
''> Настроить виджет «Переключение рабочих столов»''
''> Основное''

* Снять галочку «Показывать значки приложений на схемах окон»
* Поставить галочку «Показывать только текущий экран» (это важная настройка для двухмониторных конфигураций)
* Поставить галочку «Навигация с переходом по циклу»
* Переключить радиобаттон «При выборе текущего рабочего стола: Не делать ничего»

===== Настроить блокировку экрана =====
'''
Параметры системы 
> Рабочая среда 
> Поведение рабочей среды 
> Блокировка экрана
'''
* Автоматически блокировать экран: При отсутствии активности в течении 10 минут.

===== Настроить окно прогресс-бара при копировании/удалении =====
Мне не нравится отображение прогресс-бара в трее, хочу видеть это в отдельном окне. 

'''
Параметры системы 
> Предпочтения пользователя
> Уведомления
> Ход выполнения
'''
* Убрать галочку «Показывать в диспетчере задач»
* Поставить галочку «Показывать в уведомлениях»
* Поставить галочку «Не закрывать всплывающее окно во время выполнения»

'''
> Заголовки уведомлений
'''
* Убрать галочку «Показывать в диспетчере задач»






The setting can be edited by:
Left clicking the 'Show hidden icons' arrow.
Right clicking the 'Notifications'
Left clicking the ' Notification settings'

Since Plasma 5.6.4 (Kubuntu 16.04) I have to disable both:

1
Параметры системы > Уведомления.

Трэй — раскрыть список "Показать скрытые значки" 
/ правой кнопкой мыши по «Системные уведомления» / выбрать «Настроить виджет "Системные уведомления"» 
/ отжать опцию «Показывать ход выполнения операций с файлами и других заданий»
/ OK

В пятой плазме этой настройки может не быть.

2
На тулбаре правой кнопкой / Настроить виджет "Панель задач" / 
* отжать галочку "Показывать в кнопках состояние и ход выполнения операций"
* отжать галочки "Показывать всплывающие подсказки" и указать, что окна не надо группировать.
* отжать галочку "Выделять окна из остальных" - отключим превью окон при наведении на их заголовки в тулбаре, бо бесит.
/ OK

Notifications / right click - Notifications Settings... / uncheck Track file transfers and other jobs / OK
right click on the task bar / Task Manager Settings.../ uncheck Show progress and status information in task buttons / OK
Then logout and login.
