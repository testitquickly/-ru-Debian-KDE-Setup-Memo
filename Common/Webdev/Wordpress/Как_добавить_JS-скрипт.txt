====== Как добавить JS-скрипт ======

< @wordpress @js >

Можно закинуть js-файл по ftp на сервер с WP, но исполняться он не будет. Надо «зарегистрировать» его внутри WP.

1. Добавить плагин «Simple Custom CSS and JS».
2. Активировать его. Если в админке русский язык, плагин будет называться «Произвольные CSS и JS».
	a. Если на сайте есть плагины, управляющие кэшем (вроде «W3 Total Cache» или «WP Fastest Cache»), вероятно, сперва надо будет сбросить кэш.
	b. Убедиться в том, что на сервере есть каталог „wp-content/uploads/custom-css-js” и он открыт на запись
3. В админке WP пройти в Плагины > «Произвольные CSS и JS» > Добавить произвольный JS
4. Обязательно добавить заголовок нового скрипта
5. Вставить текст скрипта
6. Выбрать «Тип связывания» — «Внешний файл»
7. Отредактировать «Permalink» — переименовать во что-то внятное
8. [Опубликовать]

Скопировать «Permalink», вставить в html-файл инструкцию по подключению js-файла через «Permalink». Обычное место — между </style> и </head>:

''</style>''
''<script type="text/javascript" src="Мой-Permalink-Полностью''[[">|''">'']] ''</script>''
''</head>''
