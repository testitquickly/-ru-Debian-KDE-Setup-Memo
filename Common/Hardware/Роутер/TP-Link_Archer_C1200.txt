====== TP-Link Archer C1200 ======

< @hardware @wi_fi @tp_link >

===== Настройка постоянных IP =====

Постоянный IP можно назначить для любого устройства. 

Для одного и того же устройства роутер выдает разные динамические IP при подключении через провод и через WI-Fi. Поэтому нельзя назначить постоянный IP непосредственно для ноута, с которого было сделано подключение к роутеру через Wi-Fi, сперва надо подключить ноут к роутеру проводом. 

''Дополнительные настройки''
''> Сеть''
''> DHCP-сервер''
''> Резервирование адресов''
''> Добавить''

Обязательно отметить галочку «Включить».

Мак-адрес устройства, для которого надо прописать выделенный IP (вроде ''30-03-C8-38-EE-87''), можно увидеть или на самом устройстве, или на той же странице «DHCP-сервер» в таблице  «Список клиентов DHCP».
