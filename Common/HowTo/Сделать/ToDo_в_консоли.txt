====== ToDo в консоли ======

[ @console @todolist @cheat ]

Простой способ добавить напоминалку о чём-то в консоль:

1. Сделать «под рукой» простой текстовый файл (можно и без расширения)

''mcedit ~/todo.txt''

2. Добавить в него одну или две строки о том, что надо сделать. Обязательно оставить пустую строку в конце.
3. Добавить в конец файла ''~/.bashrc'' инструкцию вызова «файла с туду» на экран

''cat ~/todo.txt''

Теперь при открытии каждого окна Konsole (или при логине в консоль в другом tty; или при перечитывании «''source ~/.bashrc''») первые строки вывода в консоли будут заполнены содержимым из ''~/todo.txt''

===== Добавить новую строку в todo =====

Всегда проще сделать

''mcedit ~/todo.txt''

но можно и так:

''echo 'Удалить контейнер Photoprism' >> ~/todo.txt''

Внимание к символам: 
''>'' — уберет из файла всё содержимое и добавит в него новую строку
''>>''  — оставит содержимое файла и добавит новую строку (в конце файла)

Текст для ''echo'' можно не обрамлять кавычками, но лучше да, чтобы привычки не нарушать.

===== Удалить определенную строку =====

Удалить, например, третью строку из файла с туду:

''sed -i '3d' ~/todo.txt''

===== Сделать краткие команды =====

Добавить в ~/.bashrc новые функции.

''mcedit ~/.bashrc''

Добавить в файл:

	''# Добавить строку в файл ~/todo.txt''
''todo() {''
	''echo "$*" >> ~/todo.txt''
''}''

	''# Удалить строку из файла ~/todo.txt''
''todorm() {''
	''sed -i "${1}d" ~/todo.txt''
''}''
	''# Алиасы для ошибок при наборе''
''alias туду='todo'''
''alias ещвщ='todo'''
''alias тудурм='todorm'''
''alias ещвщкь='todorm'''

Сохранить, применить изменения:

''source ~/.bashrc''

Теперь 

	1) добавить задачу в todo.txt:

''todo Положить диафильмы в коробку''

И даже кавычки не нужны.

	2) удалить строку/задачу из todo.txt (например, третью):

''todorm 3''
