====== locate ======

< @debian @files @search >

Мощная утилита для поиска файлов, работает как краулер, раз в сутки по крону (/etc/cron.d/find) запускает поиск по файлам и сохраняет информацию в свою базу данных (/etc/updatedb.conf).

Обычно нужна для [[Krusader]].

Ручное обновление БД:

''updatedb''

===== Поиск файлов =====

''locate abc.txt''

Поиск с игнорированием строчных или заглавных букв:

''locate -i abc.txt''
