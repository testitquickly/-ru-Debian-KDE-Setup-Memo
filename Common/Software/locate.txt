====== locate ======

[ @debian @files @search ]

Мощная утилита для поиска файлов, Обычно нужна для [[Krusader]], но вполне может использоваться в быту.

Работает как краулер, раз в сутки по крону (/etc/cron.d/find) запускает поиск по файлам и сохраняет информацию в свою базу данных (/etc/updatedb.conf) — поэтому работает быстро. Но это также означает, что поиск происходит по базе, а не по диску. Файлы на диске могут отсутствовать.

===== Искать только среди существующих файлов =====

locate -e *.txt

===== Ручное обновление БД =====

''updatedb''

===== Поиск файлов =====

''locate abc.txt''

Поиск с игнорированием строчных или заглавных букв:

''locate -i abc.txt''

==== Ограничить вывод списка найденных файлов ====

''locate abc.txt | less''

''locate -n 10 *.txt''

==== Поиск по расширению файлов ====

''locate *.txt''

==== Поиск без учета регистра буква ====

''locate -i abc.txt''

==== Посчитать количество найденных файлов ====

locate -c ''abc.txt''

==== Комплесный поиск ====

Делается через -r (--regexp).

Поиск всех .mp4 А ТАКЖЕ .avi файлов:

''locate --regex -i "(\.mp4|\.avi)"''
