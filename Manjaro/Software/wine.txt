Ставим всё сразу

sudo pacman -S wine winetricks wine-mono wine_gecko

wine is obviously the main one, but the rest are also recommended. winetricks is used to automatically install dependencies that certain Windows programs may rely on, wine-mono is for applications that rely on .NET, and wine-gecko is for applications that rely on Internet Explorer. Installing those extra packages isn’t necessary, but it may save you a headache later on.

Не надо ставить 

$ sudo pacman -S wine-staging

это development branch, он нестабилен и при установке потребует удалить wine. 

===== Настройки =====

Запустить Winetricks

Select “create new wineprefix” and click OK.

«Создать новый путь wine»



The Wine configuration menu can be accessed with this command:

$ winecfg

To access Wine’s registry editing tool, execute this command:

$ regedit

To access Wine’s version of the Windows Control Panel, enter this command to launch it:

$ wine control


===== open the Wine configuration =====

winecfg

Настроить вывод звука и размер шрифта.

To access Wine’s registry editing tool, run the following command:

Regedit

===== Установить софт через Wine =====

Какой-то (базовый) софт можно установить через Winetricks > Winetricks will ask what we’d like to do. Select “install an application”

Обычно же ставят ПО в консоли, в каталоге с файлом exe

''sudo wine file.exe''

Но лучше так: 

''wine control''
''> Установка/удаление программ''
''> Приложения'' 
''> Установить''

При установке выбрать «Тип файлов» = exe

Во время установки ПО будет предложена установка в “C:\Program Files (x86)\…”. Не надо туда ничего ставить. Надо подняться на уровень выше и поставить ПО в “C:\Program Files\…” — только тогда в меню KDE появится пункт вызова ПО из wine.

===== Дополнительный софт =====

Иногда какое-то ПО требует установить DirectX или Visual или ещё что-то подобное. Бывает, что под Wine это всё подтягивается само и работает как надо. А бывает, что нет. Если нет, то лучше уже поднять виртуальную систему, чем настраивать wine на работу с такими дополнениями.
