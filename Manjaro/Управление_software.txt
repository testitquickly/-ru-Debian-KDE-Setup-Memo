====== Управление software ======

===== Основы управления ПО =====

==== pacman ====

Основной консольный package manager, соединен с репозиторием Manjaro:

$ sudo pacman -S <package name>
$ sudo pacman -Syyu
$ sudo pacman -Rs <package name>
$ sudo pacman [-Ss] [-Qs] [-F] <package name>

==== pamac ====

Это и GUI, и консольный package manager, который соединен и c репозиторием Manjaro, и с AUR, Flatpak и Snaps:

$ pamac install <package name>
$ pamac upgrade
$ pamac remove <package name>
$ pamac search <package name>

===== Установить ПО =====

''sudo pacman -S audacity''

==== Удалить ПО - без удаления зависимостей ====

''sudo pacman -R audacity''

==== Удалить ПО - со всеми зависимостями ====

''sudo pacman -Rs audacity''
