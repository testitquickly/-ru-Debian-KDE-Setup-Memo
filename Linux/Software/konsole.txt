====== konsole ======

< @konsole @console @fonts >

Настройка 
''> Управление профилями'' 
''> Создать новый профиль с названием 'astenix'''
''> Поставить напротив этого профиля галочку в колонке "Показывать в меню"'' 
''> Поставить галочку «Профиль по умолчанию»''
''В поле «Команда» по-умолчанию может быть путь к оболочке zhs. Мне привычнее bash, поэтому:'' [[/bin/bash|''/bin/bash'']]
''Остальные настройки по своему усмотрению.''
''[Применить]''

Если Konsole запускает другой профиль, нежели заданный, можно сделать ярлык для запуска с явным указанием профиля:

''konsole --profile astenix''

Все другие профили можно удалить. 

Настройка 
'''
> Настроить профиль (astenix) 
> Внешний вид 
> шрифт IBM Plex Mono 17 (or Hack) (+сглаживать шрифты)
'''

===== Помощь в консоли =====

Стандартный мануал (если есть), выход из него через клавишу 'q':

''man ping''

Вызов короткой справки прямо в консоль:

''ping -h''
