====== pgAdmin ======

< @pgadmin >

[[Debian:Установка ПО:pgAdmin]]
[[Manjaro:Установка ПО:pgAdmin]]

Дальше согласно документации — https://www.pgadmin.org/docs/

Можно выставить интерфейс на русском языке, можно на английском:

File
> Settings
> Miscellaneous (Разное)
> User Language (Язык пользователя)

Из неочевидного: 

* после подключения к удалённой БД надо открыть «Query Tool» («Запросник») — скрыт за правой кнопкой по названию базы.
* увидеть перечень таблиц в базе: БД > Схемы > Таблицы
* настроить автодополнение:
	Файл
	> Настройки
	> Запросник
	> Autocompletition
		Включить 
		* Autocomplete on key press
		* Keywords in upper case
