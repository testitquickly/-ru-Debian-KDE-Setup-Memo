====== KeePassXC ======

< @KeePassXC >

Лучший open-source password manager. Форк умершего “KeePass”. Работает и под Windows.

Для андроид есть ''Keepass2Android'' — если положить файл с базой в Dropbox, то обновление файла будет происходить на лету. При возникновении нестыковок — записывает изменения в новую копию файла в Dropbox.

Надёжный, сбоев ещё не было. 

===== Установка =====

Для Debian: 

''sudo apt-get install keepassxc''

Другие варианты (flatpak, appimage и др): https://keepassxc.org/download/#linux
