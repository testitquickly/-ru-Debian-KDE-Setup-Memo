====== Создать новый файл ======

< @console >

===== Стандартный способ =====

''> 123.txt''

Это перенаправление вывода из консоли (с экрана) в файл. Если файл отсутствует, он будет принудительно создан. В данном случае выводить нечего, поэтому появится пустой файл с названием 123.txt

===== Нестандартный способ =====

''touch 123.txt''

Команда ''touch'' должна «потрогать» существующий файл и обновить ему время последнего доступа, якобы «файл только что открывали». Но если указанный «для трогания» файл отсутствует, то он будет создан. То есть, для ''touch'' создание файлов — косвенный эффект.

Чтобы не создавались новые файлы, надо указать ''touch'' перечень «трогаемых» файлов с ключом -c.

Через ''touch'' можно одной командой создать сразу много файлов, указав их имена последовательно:

''touch 123.txt 345.doc hello.js viva.txt''
