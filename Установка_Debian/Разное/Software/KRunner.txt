===== Другой вариант настроек =====
Параметры системы / Рабочая среда = Диспетчер окон / Особые параметры окон / Создать
"Выбор окон" / 
* Описание = krunner centering
* Класс окна (приложение) = Точное совпадение = krunner
"Геометрия" / 
* Расположение = Применить изначально = 700,250
[Применить]

Alt+F2 запускает окно KRunner.

===== Настроить поиск файлов в KRunner =====

* Параметры системы / Рабочая среда = Поиск / Поиск файлов / "Включить службу поиска файлов"
* Указать папки, в которых не надо индексировать файлы. В терминале команда balooctl status проверяет саму службу индексирования

Параметры системы / Рабочая среда = Поиск / Поиск в Plasma / оставить следующие галочки:
* Документы
* Закладки
* Запуск программ
* Папки
* Последние Документы
* Приложения
* Расположения
* Точки входа
