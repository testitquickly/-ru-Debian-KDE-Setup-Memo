====== KMyMoney ======

Адекватная, старинная аппликуха для учёта финансов. Learning curve может быть тяжеловата. Аналогов море.

''sudo aptitude install kmymoney''

Один из первых вопросов при настройке — какой формат хранения данных выбрать. Ответ: XML

===== Как зашифровать файл ключом GPG =====
Защита бывает разная. 

Шифрование с помощью личного ключа GPG превращает содержимое файла *.kmy в нечитаемую абракадабру. Зашифрованный файл можно безопасно положить в Dropbox или даже отослать по почте — даже если учётную запись в Dropbox взломают, без gpg-ключа и его пароля ничего просто так не откроется (в принципе всегда всё можно открыть, дело только в мастерстве, знаниях и ресурсах). Но эта мера защитит только когда злой умышленник этот файл *.kmy заберёт и откроет где-то на своём компьютере. 

На моём ноутбуке этот же зашифрованный файл *.kmy будет открываться СРАЗУ, безо всяких паролей, потому что gpg-ключ тут прописан в системе, и ему присвоен высочайший уровнь доверия. И если злой умышленник заберёт мой ноутбук и сумеет обойти защиту профиля (это сложно, но возможно), то содержимое файла будет доступно ему полностью.

Если надо защитить файл ВООБЩЕ, то надо использовать что-то вроде VeraCrypt — будет создан файл-контейнер, в котором будет спрятан файл *.kmy, а снаружи будет длиннопарольная защита или даже ключ с внешнего usb-диска или даже подпись gpg-ключом.

Если всё-таки шифрования с gpg достаточно, то:
1. Разобраться с созданием ключа [[GPG]], вооружиться ключом и паролем к нему.
2. ''KMyMoney ''
''> Настройка ''
''> Настроить KMyMoney ''
''> Plugins ''
''> XML Storage ''
''> настройки данного плагина ''
''> Use GPG Encryption ''
''> выбрать из выпадающего списка нужный ключ (если их несколько) ''
''> ОК ''
''> Файл ''
''> Сохранить (подтвердить понимание риска потери данных если ключ потеряется) ''
''> Файл ''
''> Выход''
3. Заново открыть файл *kmy
Поскольку ключ уже существует в системе, расшифровка произойдёт незаметно. Со временем вполне можно забыть о том, что был ключ и пережить интересный сеанс ужаса от невозможности открыть файл с ценной информацией.
