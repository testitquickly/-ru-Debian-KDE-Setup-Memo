====== Профили в Firefox ======

< @firefox @user @profile >

Есть официальная [[https://support.mozilla.org/en-US/kb/profile-manager-create-remove-switch-firefox-profiles|справка]] по всему этому делу.

===== Создать новый профиль =====

	Открыть url „about:profiles” 
	> [Создать новый профиль] 
	> дать ему внятное имя, например, “''deserviciu''”

Профили создаются в /~/.mozilla/firefox в виде каталогов с инопланетным названием. Если профиль надо будет перемещать на другие компьютеры, разумно указать для него отдельный каталог на диске. 

Каждый новый профиль помечается как «по-умолчанию», поэтому надо найти свой основной профиль и отметить его «as default». Обычно это профиль с названием «default-release».

В новый профиль можно переместить файлы из уже существующего — должны подтянуться все настройки. Учесть, что будет скопировано всё, и куки и кэш. Считается разумным делать это через механизм экспорта/импорта профилей, но можно просто скопировать всё с затиранием существующих файлов.

===== Запуск браузера с выбором профиля =====

''firefox -P''

Откроется окно с перечнем профилей.

В окне выбора профилей можно отметить галочкой один из профилей, которые в будущем будет загружаться автоматически. Это аналог назначения профиля «as default» на странице „about:profiles”.

===== Запуск браузера с определенным профилем =====

Можно сделать ярлык для запуска определенного профиля Firefox:

''firefox -p deserviciu''

Если это рабочий аккаунт, то лучше назначить этот профиль дефолтным. 

Иногда безопаснее использовать для работы и для личного не разные профили, а разные браузеры.
