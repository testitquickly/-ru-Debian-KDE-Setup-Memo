====== qmmp ======

Самая удобная обложка: //iAmp1-6.wsz//

===== Размер плеера =====

Общие настройки (правой кнопкой по титлебару qmmp)
'''
> Вид 
+ Двойной размер (Meta+D)
+ Сглаживание
'''

===== Включить вкладки в плейлисте =====

Общие настройки
'''
> Список
+ Повторять список
+ Показывать вкладки
+ Группировать треки
'''

Группировка треков очень опциональна.

==== Управление вкладками ====

В интерфейсе qmmp они называются «Список» и представляют собой m3u файлы, которые можно сохранять на hdd по-отдельности.

В норме долна быть возможность включить отображение кнопки добавления новой вкладки.

Настройки  (Ctrl+P)
'''
> Внешний вид
> Список песен 
> Показывать кнопку "Новый список"
'''

Там же настраивается вид разделителя списков.  

	В qmmp версии ''2.1.2'' у меня эта настройка неактивна. 

В любом случае управление списками делается элементарно из меню управления списками в правом нижнем углу плейлиста, или же шорткатами:

* новый список: ''Ctrl+T''
* удалить список: ''Ctrl+W''
* переименовать список: ''F2''
* переключение между списками: ''Ctrl+PgUp/PgDown''
* показать все списки (и управлять ими): ''P''

===== Настройки модуля http =====

Настройки  (Ctrl+P)
'''
> Модули
> Транспорты 
> Модуль HTTP 
> Настройки
'''
		* Размер буфера = 200 Кб
		* Кодировка по умолчанию = UTF-8

===== Настройка модуля CUE =====

Настройки
'''
> Модули
> Декодеры
> Модуль CUE
> Настройки
> Основные настройки
'''
		+ Загружать Cue с ошибками, если возможно
		Это помогает загружать плейлист даже если что-то недоступно.
'''
> Кодировка
'''
		+ Кодировка по умолчанию = UTF-8

===== Настройка встроенной регулировки громкости =====

Настройки 
''> Аудио''
''> Аудио''
	+ Использовать программную регулировку громкости

Это включает отдельное управление громкостью ползунком на qmmp, иначе управление громкостью передаётся на уровень всего KDE.

===== Настройка модуля глобальных клавиш =====

Настройки 
''> Общие'' 
''> Модуль глоб. клавиш''
	* Воспроиз/приост = Shift+Alt+d
	* Следующий фрагмент = Shift+Alt+Next
	* Предыдущий фрагмент = Shift+Alt+Prior

Остальное по усмотрению. 

===== Несущественное =====

==== Настройка модуля записи файлов ====

Обратить внимание особо:

Настройки
'''
> Модули
> Эффекты
> Модуль записи файлов
'''
	Не включать без нужды, иначе qmmp будет постоянно записывать ВСЁ, что проигрывается, в файлы .ogg в каталог Музыка.

==== Настройка распознавания CD-дисков ====

Настройки
'''
> Модули
> Общие
> Модуль UDisks
'''
	Нужен, если надо часто использовать музыку с CD. Он помогает автоматически распознавать музыку на дисках и добавлять её в плейлист. В остальном включать нет нужды.

===== Чего нет =====

Нет возможности управлять скоростью воспроизведения. Можно использовать аналог — [[Audacious]]
